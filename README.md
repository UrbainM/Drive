# Drive
A simulated environment which stores objects.  Once introduced to the environment the objects are no longer in physical memory until retrieval.

Working:  2-Dimensional storage and retrieval of introduced 8x8 pixel avatars.
Next:  quadruples are a placeholder, the data complexity comes from layers of n-wise pairs independently examinable by a quasiphysical process.
